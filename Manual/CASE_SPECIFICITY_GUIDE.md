## 🎯 개선 목표

### Before (간결형)
```
30대 남성이 회의 중 쓰러졌다.
맥박은 촉지되지 않는다.
심전도: [Image: "편평한 파형" - "무수축"]
```

**특징:**
- ✅ 핵심만 간결
- ⚠️ 현장감 부족
- ⚠️ 실전 감각 제한

### After (현장 재현형)
```
오후 2시, 30대 남성이 회사 회의실에서 프레젠테이션 중 갑자기 쓰러졌다.
동료 김OO가 즉시 119에 신고(14:03)하고 CPR을 시작했다.
구급대 도착(14:09, 6분 소요) 시 맥박 촉지 안 됨, 호흡 없음, 피부 창백.
과거력: 고혈압 약물 복용 중.
심전도: [Image: "편평한 파형" - "무수축"]
```

**특징:**
- ✅ 실제 현장과 유사
- ✅ 상황 판단력 향상
- ✅ 몰입감 UP
- ⚠️ 약간 길어짐 (허용 범위)

## 📝 구체화 요소 (Option 1: 기본 구체화)

### **1. 시간/날씨 정보**

**목적:** 상황의 긴급성과 환경 조건 제공

**예시:**
```
✅ "오후 3시, 무더운 여름날 (외기온 35°C)"
✅ "한겨울 새벽 (외기온 -10°C)"
✅ "장마철 오전 10시, 습도 높은 날"
✅ "늦은 밤 11시, 퇴근 시간대"

❌ "2025년 7월 20일 14시 32분" (너무 구체적)
❌ "화창한 봄날 벚꽃이 흐드러진" (불필요)
```

### **2. 장소 구체화**

**목적:** 현장 환경과 접근성 정보 제공

**예시:**
```
✅ "피트니스센터 러닝머신에서"
✅ "북한산 백운대 코스 6부 능선에서"
✅ "강남역 2번 출구 앞 횡단보도에서"
✅ "회사 지하 주차장에서"

❌ "서울시 강남구 테헤란로 427 4층" (과도)
❌ "총 면적 500평의 대형 헬스장" (불필요)
```

### **3. 발견 상황**

**목적:** 초기 대응자와 조치 제공

**예시:**
```
✅ "동료가 즉시 119 신고 및 CPR 시작"
✅ "가족이 발견 후 5분 뒤 구급대 도착"
✅ "현장 목격자가 AED 사용 시도"
✅ "경비원이 발견, 초기 조치 없음"

❌ "홍길동(45세, 남성)이 발견하여..." (과도)
```

### **4. 활력징후 (2-3가지)**

**목적:** 임상적 판단 근거 제공

**우선순위:**
1. **맥박** (심정지 확인)
2. **호흡** (심정지 확인)
3. **혈압** (순환 상태)
4. **체온** (저체온증 등)
5. **산소포화도** (호흡 상태)

**예시:**
```
✅ "맥박 촉지 안 됨, 호흡 없음, 혈압 측정 불가"
✅ "혈압 85/50mmHg, 맥박 110회/분, 산소포화도 88%"
✅ "체온 27°C, 맥박 촉지 안 됨, 호흡 없음"

❌ "혈압 120/80, 맥박 72, 호흡 16, 체온 36.5, SpO2 98%, 혈당 110" (과도)
```

### **5. 외관/임상 소견**

**목적:** 시각적 정보와 중증도 제공

**예시:**
```
✅ "피부 창백하고 땀으로 젖어 있음"
✅ "청색증 관찰, 입술과 손톱이 파랗게 변색"
✅ "식은땀 흐르며 얼굴이 일그러짐"

❌ "키 175cm, 체중 70kg, BMI 22.8" (불필요)
```

### **6. 간단한 병력 (필요 시)**

**목적:** 가역적 원인 또는 리스크 요인 제공

**예시:**
```
✅ "고혈압 약물 복용 중"
✅ "만성 신부전으로 주 3회 혈액투석"
✅ "3일 전 무릎 수술 후 침상 안정 중"
✅ "장거리 비행(12시간) 직후"

❌ "10년 전 충수절제술, 5년 전 백내장 수술..." (과도)
```

## 📐 **구성 원칙**

### **3-5문장 원칙**
```
문장 1: 시간/날씨 + 연령/성별 + 장소
문장 2: 발견 상황 + 초기 조치
문장 3: 활력징후 2-3가지
문장 4: (선택) 외관 또는 병력
문장 5: 심전도
```

### **균형 유지**
```
구체화 60% + 간결성 40% = 최적
```

## 📊 **기대 효과**

### **1. 교육적 가치 향상**
```
Before: 단순 지식 확인
After:  실제 상황 판단력 훈련
```

### **2. 몰입감 증가**
```
Before: "또 하나의 문제"
After:  "실제 현장 같은 느낌"
```

### **3. 맥락 기반 추론**
```
- "무더운 날 + 마라톤" → 탈수/열사병 고려
- "겨울 등산 + 저체온" → 재가온 우선
- "수술 후 + PEA" → 폐색전증 의심
```

### **4. 다양성 자동 증가**
구체화하면서 자연스럽게 다양해짐:
- 시간대: 새벽, 오전, 오후, 저녁, 밤
- 날씨: 여름, 겨울, 장마철
- 상황: 운동, 업무, 등산, 수영 등

### **❌ 하지 말아야 할 것**

1. **과도한 상세화**
```

❌ "회사는 서울시 강남구 테헤란로 427번지에 위치한..."
```

2. **불필요한 정보**
```
❌ "환자는 10년 전 충수절제술, 5년 전 백내장 수술 받음"
   (심정지와 무관)
```

3. **핵심 흐리기**
```
❌ 병력이 너무 많아서 심전도가 묻힘
❌ 상황 설명이 길어서 질문을 놓침
```

### **✅ 해야 할 것**

1. **핵심 중심 구체화**
```
✅ 심전도 판독에 도움이 되는 정보
✅ 처치 결정에 영향을 주는 배경
✅ 가역적 원인 힌트 (PEA/무수축일 때)
```

2. **적절한 길이**
```
✅ 3-5문장 (60-100단어)
✅ 읽는 데 10-15초
```

3. **몰입감 UP**
```
✅ "갑자기 쓰러졌다" → "가슴을 움켜쥐고 쓰러졌다"
✅ "맥박 없음" → "목동맥과 요골동맥 모두 촉지 안 됨"
✅ "CPR 시작" → "동료가 즉시 가슴압박과 인공호흡 시작"
```
