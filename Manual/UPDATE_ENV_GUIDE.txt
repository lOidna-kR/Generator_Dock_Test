===============================================================
.env 파일 수정 가이드
===============================================================

현재 설정 문제:
1. RETRIEVAL_K=20 → 너무 많음 (7로 변경 필요)
2. LLM_TEMPERATURE=1.4 → 너무 높음 (0.85로 변경 필요)
3. RETRIEVAL_INITIAL_K 없음 → 20으로 추가 필요

===============================================================
수정 방법
===============================================================

1. .env 파일을 에디터로 열기

2. 다음 라인을 찾아서 수정:

[현재]
RETRIEVAL_K=20
LLM_TEMPERATURE=1.4

[수정 후]
RETRIEVAL_INITIAL_K=20
RETRIEVAL_K=7
LLM_TEMPERATURE=0.85

3. MCQ 설정 추가 (파일 끝에):

MCQ_MAX_CONTEXT_DOCS=7
MCQ_FEW_SHOT_MAX_EXAMPLES=2

4. 저장

===============================================================
변경 사항 요약
===============================================================

검색 흐름:
  초기 검색 → 리랭킹 → LLM 전달
  20개     → 7개      → 7개

설정:
- RETRIEVAL_INITIAL_K=20  (초기 검색)
- RETRIEVAL_K=7           (리랭킹 후)
- MCQ_MAX_CONTEXT_DOCS=7  (LLM 전달)
- LLM_TEMPERATURE=0.85    (창의성)
- MCQ_FEW_SHOT_MAX_EXAMPLES=2 (다양성)

===============================================================
검증 방법
===============================================================

수정 후 다음 명령으로 확인:
  python test_config.py

기대 출력:
  initial_k (초기 검색): 20
  k (리랭킹 후): 7
  max_context_docs (LLM 전달): 7
  Temperature: 0.85
